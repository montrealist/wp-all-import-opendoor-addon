Easily import property listings from any XML or CSV file to the WP Open Door theme with the WP All Import Open Door Theme Add-On for WP All Import.

== Description ==

The WP All Import Open Door Theme Add-On for [WP All Import](http://wordpress.org/plugins/wp-all-import "WordPress XML & CSV Import") makes it easy to bulk import your property listings to the WP Open Door theme.

[Read more about getting an Add-On made for your plugin or theme.](http://www.wpallimport.com/add-ons/#developers)

== Installation ==

First, install [WP All Import](http://wordpress.org/plugins/wp-all-import "WordPress XML & CSV Import").

Then install the WP All Import Open Door Theme Add-On.

To install the WP All Import Open Door Theme Add-On, either:

* Upload the plugin from the Plugins page in WordPress

* Unzip wp-opendoor-add-on-for-wp-all-import.zip and upload the contents to /wp-content/plugins/, and then activate the plugin from the Plugins page in WordPress

The Add-On will appear in the Step 3 of WP All Import.
